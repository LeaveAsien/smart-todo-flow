# My Blog 开发计划

> This is an example PLAN.md showing how to structure project goals into phases.

## Phase 1 — 基础搭建

1. 项目初始化（Next.js + TypeScript + Tailwind）
2. 首页文章列表：从本地 Markdown 文件读取，按日期倒序展示
3. 文章详情页：Markdown 渲染 + 代码高亮
4. 基础布局：导航栏、页脚、响应式适配

## Phase 2 — 功能扩展

5. 标签系统：文章支持多标签，标签页可筛选
6. RSS 订阅：自动生成 feed.xml
7. 搜索功能：前端全文搜索（Fuse.js）
8. 暗色模式：跟随系统 + 手动切换
9. SEO 优化：Open Graph、sitemap、结构化数据

## Phase 3 — 部署上线

10. Vercel 部署 + 自定义域名
11. 性能优化：图片懒加载、字体优化
12. 写 3 篇种子文章
