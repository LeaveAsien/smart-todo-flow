# My Blog Development Plan

> This is an example PLAN.md showing how to structure project goals into phases.

## Phase 1 — Foundation

1. Project setup (Next.js + TypeScript + Tailwind)
2. Homepage post list: read local Markdown files, display in reverse chronological order
3. Post detail page: Markdown rendering + code highlighting
4. Base layout: navbar, footer, responsive design

## Phase 2 — Feature Expansion

5. Tag system: posts support multiple tags, tag page for filtering
6. RSS feed: auto-generate feed.xml
7. Search: client-side full-text search (Fuse.js)
8. Dark mode: follow system preference + manual toggle
9. SEO: Open Graph, sitemap, structured data

## Phase 3 — Deploy & Launch

10. Vercel deployment + custom domain
11. Performance: lazy-load images, font optimization
12. Write 3 seed posts
